## Proje Özeti ve Vizyon

Günümüzün rekabetçi iş dünyasında, işletmeler için zaman ve müşteri yönetimi hayati bir öneme sahiptir. CepTakvim, güzellik salonları, klinikler, kuaförler ve randevu ile çalışan tüm işletmeler için özel olarak geliştirilmiş; defter-kalem karmaşasını tamamen ortadan kaldıran modern bir İşletme Yönetim Sistemi (Mini ERP)'dir.

Piyasadaki hantal ve öğrenme eğrisi yüksek yazılımların aksine CepTakvim, çift taraflı kusursuz bir deneyim sunar: Müşterilere 7/24 pürüzsüz bir online randevu alma sihirbazı sağlarken, işletme sahiplerine personelden ciroya, CRM'den takvim yönetimine kadar her şeyi tek bir ekrandan kontrol etme gücü verir. Güçlü MERN Stack mimarisi, dinamik SEO altyapısı ve otomatik bildirim motoruyla CepTakvim; sıradan bir takvim uygulaması değil, işletmeleri büyüten sadık bir dijital asistandır.

## ✨ Temel Özellikler (Key Features)

 🌟 7/24 Online Randevu Sihirbazı: İşletmelere özel oluşturulan dinamik profiller (/isletme/slug) üzerinden müşterilerin kolayca hizmet, personel, tarih ve saat seçerek sürtünmesiz randevu alabilmesi.

📊 Gelişmiş Yönetim Paneli (Dashboard): İşletme sahipleri için günlük ciro, yaklaşan randevular, bekleyen onaylar ve personel istatistiklerini gerçek zamanlı sunan güvenli arayüz.

🔔 Akıllı Bildirim & Hatırlatıcı Motoru: Nodemailer altyapısı ile yeni randevu, onay ve iptallerde anında e-posta gönderimi. Müşteri "no-show" (gelmeme) oranlarını sıfıra indiren otomatik hatırlatma kurguları.

👥 Müşteri İlişkileri Yönetimi (CRM): Randevu alan her müşterinin geçmiş işlemlerini, harcamalarını ve onlara dair özel notları (Örn: Kahvesini nasıl içer?) saklayan sadakat artırıcı veritabanı.

💳 Iyzico ile Güvenli Ön Ödeme (Kapora): Sahte randevuları ve randevuya gelmeme durumlarını engellemek amacıyla, online randevu sırasında kredi kartı ile güvenli ön ödeme (kapora) tahsilatı.

🔍 Dinamik ve Çift Yönlü SEO Mimarisi: Vitrin ve blog sayfalarında arama motoru optimizasyonunu (Google görünürlüğünü) maksimize eden, aynı zamanda Dashboard gibi işletme verisi içeren gizli sayfaları botlardan koruyan (noindex) akıllı yapı.

📱 Duyarlı ve Adaptif UI (Responsive Design): Hem müşteriler hem de işletme sahipleri için mobil, tablet ve masaüstü cihazlarda kusursuz çalışan modern, akıcı ve şık arayüz.

## ⚙️ Kurulum Rehberi (Installation)

Projeyi yerel ortamınızda çalıştırmak ve geliştirmeye başlamak için aşağıdaki adımları sırasıyla izleyebilirsiniz:

1. **Projeyi Klonlayın**
```bash
git clone https://github.com/oguzhan1414/CepTakvim-Projesi.git
cd CepTakvim-Projesi
```

2. **Backend (Sunucu) Kurulumu ve Başlatılması**
```bash
cd backend
npm install
# Lütfen projeye uygun bir .env dosyası oluşturarak MongoDB URI ve diğer gizli değişkenlerinizi tanımlayın.
npm start
```

3. **Frontend (İstemci) Kurulumu ve Başlatılması**
```bash
cd frontend
npm install
npm start
```
